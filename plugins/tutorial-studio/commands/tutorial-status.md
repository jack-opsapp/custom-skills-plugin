# Tutorial Status

Check the current state of tutorial design work in this project.

## Search for Artifacts

1. **App Profile documents** — look in docs/ for files containing "App Profile" or "app-profile"
2. **Audit Reports** — look for files containing "Audit Report" or "audit-report"
3. **Tutorial Design Documents** — look for files matching `tutorial-design-*` in docs/
4. **Generated tutorial code:**
   - iOS: check for `Tutorial/` directory with `State/`, `Wrappers/`, `Views/` subdirectories
   - Web: check for `components/tutorial/` and `lib/tutorial/` directories
